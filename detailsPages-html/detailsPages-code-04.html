github绑定域名教程

0x00:

我在腾讯云上注册了一个域名：atyoume.com， 我已经在GitHub Pages上建立了自己的博客：http://zfcdaemon.github.io/mysite/。 现在我希望将atyoume.com映射到http://zfcdaemon.github.io/mysite/。

0x01:

创建CNAME文件夹在你的个人博客仓库的根目录中新建文件CNAME（注意没有后缀），在该文件增加一行文字， 告诉Github Pages服务器你想指定的域名。该域名不能包含前缀信息，即不能添加http:\\前缀。 (注：CNAME文件名一定要大写，否则Github Pages服务器无法识别和解析。)

0x02:

登陆到腾讯云的域名解析控制台，如下图添加相应设置（红色标记区）。

若图片看的不清楚，下面为文字叙述：

主机记录:@ -- 记录类型:A -- 记录值 : 192.30.252.153

主机记录:@ -- 记录类型:A -- 记录值 :192.30.252.154

主机记录:www -- 记录类型:CNAME -- 记录值: username.github.io.

用你自己的 Github 用户名替换 username

推荐使用 DNSPOD 的服务，使用国外的 DNS 解析服务可能有被墙的风险.

0x03:

等待你的 DNS 配置生效.

完

CSS3图片动态提示效果

box-shadow属性： 功能:用于给元素的边框添加阴影效果。  语法：box-shadow:X轴偏移量 Y轴偏移量 阴影模糊半径 阴影扩展半径 阴影颜色 投影方式。

z-index属性:功能：设置元素的上下层显示顺序。 语法：z-index:整数值。（值大会显示在上面，值小会显示在下面。）

tansform: 功能：使元素变形的属性，其配合rotate（旋转角度）,scale(缩放倍数)，skew（扭曲元素）等参数一起使用。  语法：transform:rotate(旋转角度)；transform:scale(缩放倍数)

transition: 功能：设置元素由样式1变成样式2的过程所需的时间。  语法：transition:duration。

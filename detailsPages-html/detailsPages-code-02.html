jQuery:click事件更改CSS样式+还原，及focus事件

实例：

选择或填写您要捐助的金额：

  • 500元
  • 1000元
  • 2000元
    •  元

CSS部分：

					.help-money ul li{
						float: left;
						margin-left: 40px;
						list-style: none;
						width: 90px;
						height: 30px;
						line-height: 30px;
						border: 2px #CCCCCC solid;
						border-radius: 10px 10px;
						text-align: center;	
					}
					.help-money ul li:hover{
						border: 2px #063869 solid;
					}
					.help-money ul a{
						display: inline-block;
						width: 90px;
						height: 30px;
						color:#063869;
						font-weight: 600;
						text-decoration: none;	
						
					}
					.help-money-list input{
						display: inline-block;
						width: 120px;
						height: 28px;
						color:#333333;
						font-size: 14px;
						font-weight: 600;
						line-height: 28px;
						margin-left: 40px;
						padding-left: 5px;
					}

JS部分：

					$(document).ready(function(){			
						//		捐助金额选择
						$(".donateList").click( 
						  function() {		
						        $(this).css("border","2px solid #063869").siblings().css("border","2px solid #CCCCCC"); 
						   });
						$("#other-money").focus(
							function(){
								$(".donateList").css("border","2px solid #CCCCCC"); 
							}
						)			
					});

另：Input框中的 

						onkeyup="this.value=this.value.replace(/[^\d]/g,'') "
						与 
						onpaste="return false"

						前者的正则表达式为只允许输入框中输入数字，后者为禁止在输入框中粘贴内容

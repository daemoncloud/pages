@media all and(min-width:800px)and(orientation:landscape) { .... } 解释：all--指媒体类型、（默认为all，可省略，其他类型还包括 screen或print等等。） and--逻辑操作的查询符来连接（还包括not ,or,only等。 如上述例句，在and连接中如果返回的布尔值为真，则会应用花括号里面定义的代码，反之则不应用。）
